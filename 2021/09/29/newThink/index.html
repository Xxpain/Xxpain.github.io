生活

搬家了，搬到浦东新区，离公司倒是近了不少，通勤一天一个小时，很满足，可惜就是夏天有蟑螂，
没事看到个蟑螂直接头皮发麻，有点想移民火星了😭😭😭😭😭😭，最近运动也少了，变胖不少呜呜呜呜；
戒烟三个月了，继续；

学习

  • 测试驱动开发

  • Effective java

  • 重构 改善及有代码的设计

    技术相关学习

  • python

    本来大学学了听多，不过工作一直没咋用，都忘完了，呜呜呜，又捡起来学了几天，不过最近中秋节一放假，又变成了纯纯的懒狗了😥
